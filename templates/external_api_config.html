How to Use
  1. Update API URL: Set the correct base URL for your external API
  2. Test Connection: Use a valid user ID to test if the API is accessible
  3. Access with User ID: Use URLs like /post?user=USER_ID to automatically fetch user data
  4. Manual Fetch: Use /fetch_external_user/USER_ID to manually fetch and store user data
Note: The application will automatically try to fetch user data from the external API when a user_id parameter is provided. If the API is unavailable, it will fall back to default data.
